# WinSaveBackup

**WinSaveBackup** allows you to easily **backup and restore** your latest PC game save data.
![Screenshot_2026-02-01-15-58-21-133-edit_com winlator cmod](https://github.com/user-attachments/assets/79253e0f-5e11-40c3-9f21-36ff66a63de9)

---

## Requirements

- 32-bit or 64-bit
- **Windows** or **Wine**
- Must be **run as Administrator** on Windows
- **Ajay Prefix** should **not** be installed  
  *(If it is installed, use the built-in backup/restore feature from the Ajay Prefix Start Menu instead)*

---
## Limitation

- It doesn't backup of official steam save data
- It doesn't backup save from Game folder
- It doesn't backup theses folders
-- Desktop , Start Menu , Microsoft , Music , Videos , Temp , Links , Pictures , Searches , Downloads , Contacts , Package Cache
---

*Note: Always run WinSaveBackup with elevated privileges to ensure full access to game save directories.*
